This plugin is mainly used for [LIGA: Pro Journey](https://github.com/rxpev/LIGA-Pro-Journey).

# 🎮 Features

### **Mechanical Improvements**

Bots are much stronger in every way imaginable when using BetterBotsPlus, one of them being improved aiming abilities.
- Advanced grenade usage with over 1000+ grenades on 10 maps
- Special pistol round grenades
- Retake grenades
- Advanced retake logic
- Bots fake plant / fake defuse
- Improved economy management (eco rounds, force buys)
- Improved bot movement
- Bots dropping weapons to each other
- AWPer improvements (holding angles - early game peeks)
- Bots saving AWP's for their AWPer (and dropping them AWP's when they can't afford one)

  and much more!
